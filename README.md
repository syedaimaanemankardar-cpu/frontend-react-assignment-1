# Cookify

![Cookify Logo](public/favicon.svg)

Cookify is a modern, AI-powered recipe platform designed to help home cooks of all skill levels transform available ingredients into delicious meals. With an intuitive interface, meal planning tools, and smart recipe generation, Cookify makes cooking easier, smarter, and more enjoyable.

## Features

- **Smart Recipe Generator**: Enter ingredients and get instant recipe suggestions.  
- **Meal Planner & Serving Adjustments**: Scale recipes for any number of servings.  
- **Favorites & Bookmarks**: Save and revisit your favorite recipes.  
- **Dietary Filters**: Vegetarian, vegan, gluten-free options.  
- **Time Estimates**: Know how long each recipe takes to prepare.  
- **Fresh & Healthy Options**: Nutritional guidance for balanced meals.  
- **Responsive & Aesthetic UI**: Modern design with Tailwind CSS and Lucide icons.
